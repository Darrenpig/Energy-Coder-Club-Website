# 小程序资源文件说明

## 图标文件要求
所有图标文件必须使用PNG格式，不能使用SVG格式。建议尺寸：

- 导航图标：44x44px
- 功能图标：32x32px  
- 头像图标：120x120px
- 项目封面：400x300px

## 图标替换指南

1. 将设计好的PNG图标文件放入对应目录
2. 确保文件命名与代码中引用的一致
3. 建议使用透明背景的PNG图标
4. 图标颜色应与深色主题协调

## 设计建议

- 使用线性图标风格，保持简洁
- 主色调使用 #16a34a (绿色)
- 次要色调使用 #0d9488 (青绿色)
- 背景使用深色透明效果
- 保持图标视觉一致性
